Om Olsen Data

Olsen Data er et nyoppstartet teknologiselskap som bygger enkle løsninger med tydelig effekt.

Vi gjør digitalisering forståelig, rimelig og gjennomførbar uten kompliserte systemer.

Hvorfor vi finnes

Vi tror små digitale grep kan gi store forbedringer i hverdagen.

Derfor bygger vi løsninger som automatiserer rutiner og frigjør tid til kjernearbeid.

Små programmer. Stor effekt.

Det er ikke bare slagordet vårt. Det er arbeidsmetoden vår.
